# MoneyLog

## 개요
- 쓰고 공유할수록 정확해지는 가계부 웹 애플리케이션

## 구현목표
[구현목표]
- 가계부 기록을 통한 지출 회고
- 가계부 분석으로 소비 패턴 및 비율 파악
- 가계부 공유로 다양한 의견을 들을 수 있는 플랫폼 제공
